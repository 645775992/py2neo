Represents a Relationship within a Neo4j database instance identified by a URI. This class is a subclass of IndexableResource and, as such, may also contain URIs identifying how this Relationship is represented within an Index.

Creates a representation of a Relationship identified by URI; optionally accepts further URIs representing both an Index for Relationships plus the specific entry within that Index.

Parameters:
  • uri - the URI identifying this Relationship
  • index_entry_uri - the URI of the entry in an Index pointing to this Relationship (optional)
  • index_uri - the URI of the Index containing the above Relationship entry (optional)
  • http - httplib2.Http object to use for requests (optional)
  • user_name - the user name to use for authentication (optional)
  • password - the password to use for authentication (optional)
