Represents a Node within a Neo4j database instance identified by a URI. This class is a subclass of IndexableResource and, as such, may also contain URIs identifying how this Node is represented within an Index.

Creates a representation of a Node identified by URI; optionally accepts further URIs representing both an Index for Nodes plus the specific entry within that Index.

Parameters:
  • uri - the URI identifying this Node
  • index_entry_uri - the URI of the entry in an Index pointing to this Node (optional)
  • index_uri - the URI of the Index containing the above Node entry (optional)
  • http - httplib2.Http object to use for requests (optional)
