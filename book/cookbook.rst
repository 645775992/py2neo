Escaping Values in Cypher
=========================

    >>> from py2neo.cypher import cypher_escape
    >>> rel_type = "KNOWS WELL"
    >>> statement = "CREATE (a)-[ab:%s]->(b) RETURN ab" % cypher_escape(rel_type)
    >>> statement
    'CREATE (a)-[ab:`KNOWS WELL`]->(b) RETURN ab'
