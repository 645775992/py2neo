=====================
API: Servers & Stores
=====================

.. attention:: These modules are designed to work on Linux and may not operate
               correctly - or at all - on other platforms.


The ``server`` and ``store`` modules provide classes for modelling a Neo4j server installation
on disk and its contained data store.
